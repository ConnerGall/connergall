The Bandits Plunder

April 2025

Summary

A 2D side-scroller inspired by classic action platformers. Code and gameplay systems are original; some art assets were adapted.

A 2D side scroller that was inspired by Castlevania. The assets were not mine but the code is.

Tools / Tech Used

  • Engine: Unity / C#
  • Tile Editor and Pixel Art tools (Aseprite / Photoshop)
  • Version control: Git

What I Learned

  • Designing responsive player movement and enemy AI.
  • Level design that encourages exploration.
  • Polishing feel (tuning jump, dash, and hit feedback).
